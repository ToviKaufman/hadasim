📌 README – Trip Tracking System
 תיאור כללי
מערכת לניהול ומעקב אחר טיול תלמידות בזמן אמת.

המערכת מיועדת לשלושה סוגי משתמשים:

תלמידות
מורות
אמהות
המערכת מאפשרת רישום לטיול, צפייה בפרטי המשתמשים, ומעקב אחר מיקום התלמידות בזמן אמת על גבי מפה, כולל התרעות על התרחקות מהקבוצה.

ארכיטקטורת שכבות (API / BL / DAL) מאפשרת הפרדה ברורה בין לוגיקה, גישה לנתונים ו-API

 סימולציית מיקומים
המערכת כוללת סימולטור המדמה מכשירי איכון:

עדכון מיקום כל חצי שניה

שליחת עדכון לשרת
הפצת עדכון לכל הקליינטים דרך SignalR
 Flow המערכת
משתמש מזין תעודת זהות
בדיקה האם המשתמש קיים:
אם לא → רישום

הקליינט מעדכן UI דרך Context
 ארכיטקטורה

 הוראות הרצה
1. Clone

2. Client
cd client
npm install
npm start
3. Server
לפתוח את הפתרון ב-Visual Studio
להגדיר את פרויקט ה-API כ־Startup Project
להריץ את הפרויקט

עדכון Connection String
יש לעדכן בקובץ:

appsettings.json
את השדה:

"ConnectionStrings": {
  "DefaultConnection": "Server=YOUR_SERVER;Database=TripDB;Trusted_Connection=True;"
}
